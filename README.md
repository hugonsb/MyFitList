# MyFitList
Aplicativo para te ajudar a organizar seus planos de treino e dieta.

## Tecnologias usadas e características
- Framework Jetpack Compose
- Kotlin
- Persistência de dados com o Room
- Google Fonts
- Pattern MVVM
